# 🛒 Shop App - Django

A simple **E-Commerce / Shop Web Application** built using **Django**. This project demonstrates the basics of Django development including routing, templates, views, static pages, and database integration.

## 🚀 Features

- 🏠 Home page for the shop  
- 📄 About page  
- 🛍️ Product/Service display pages  
- 📬 Contact form integration  
- 🎨 HTML templates with Django Template Engine  
- 💾 SQLite database support  


## 🛠️ Tech Stack

- **Backend:** Python, Django  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite3
